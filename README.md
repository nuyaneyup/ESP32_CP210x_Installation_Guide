# Arduino ESP32 CP210x Setup Guide / Kurulum Rehberi

Available in Turkish 🇹🇷 and English 🇬🇧.

## English
### Description
This guide explains how to set up an ESP32 board with Arduino IDE and the CP210x driver.  
It also includes a basic LED blink test.

### Contents
- Arduino IDE installation
- CP210x driver setup
- ESP32 board configuration
- COM port detection
- Blink test
